# Build and Run KMP Application

To build this app your must first:
1. Create a new project at [Google Firebase](https://firebase.google.com)
2. Register an Android (iOS) app with the package name (bundle id) `edu.gvsu.cis.kmpfirebase`
3. Download `google-service.json` to the `composeApp` folder of this project
3. Download `GoogleService-Info.plist` and use XCode "Add File to iosApp" to add the .plist file to iOS 
   portion of this KMP project


Warning: the instructor's original copy of the `.json` and `.plist` files are not included in the
sample as they are excluded by the .gitignore rules. You may see a warning when opening the iOS
project in Xcode.
